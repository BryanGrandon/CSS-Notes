## _Selector_

```CSS
/* All elements */
* {}

/* Elements HTML */
div {}

/* Attribute Class */
.class {}

/* Attribute Id */
#id {}

/* Descendent */
.parent .child {}

/* Direct descendent */
.parent > .child {}

/* Adjacent sibling */
.child + .sibling {}

/* Far sibling */
.child ~ .sibling {}

/* Have both classes */
.class1.class2 {}

/* Attribute */
[attribute-name] {}
[exact="value"] {}
[Has~="value"] {}
[end-in$="ends in"] {}
[begins-with^="begins"] {}
```
